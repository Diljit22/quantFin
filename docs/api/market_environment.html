Market Environment API Documentation¶

This section documents the MarketEnvironment class from the market package.

market_environment.py¶

Defines the MarketEnvironment class, a lightweight container for market data, currently supporting only a risk-free interest rate.

This module provides a thread-safe container for exogenous market data. Future extensions may include yield curves, volatility surfaces, etc.

class src.market.market_environment.MarketEnvironment(rate: float = 0.0)¶

Bases: object

Container for exogenous market data.

Parameters:

rate (float, optional) – The annualized risk-free rate as a decimal (e.g., 0.05 for 5%). Defaults to 0.0.

Notes

  • Additional market data (yield curves and volatility surfaces) to be added.

  • Thread-safety ensured via reentrant lock.

  • Negative rates are allowed.
